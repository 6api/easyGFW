## 一、简介

1. `easyGFW` 是一款`科学上网`一键部署工具，包含`VPN`和/或`SOCKET5`
2. 目前仅限在CentOS 6.0以上版本安装（32/64位均可）

## 二、安装和使用

1、安装方法

通过Git下载（推荐）

	yum -y install git
	git clone https://github.com/xiaosong/easyGFW.git
	cd easyGFW && ./install.sh
